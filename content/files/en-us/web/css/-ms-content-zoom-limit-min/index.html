The -ms-content-zoom-limit-min CSS property is a Microsoft extension that specifies the minimum zoom factor.

{{cssinfo}}

Syntax

Values

{{cssxref("percentage")}}

The minimum zoom factor.

Formal syntax

  {{csssyntax}}

Specifications

Not part of any specification.

Remarks

This property constrains the limit for touch zooming as well as values of the msContentZoomFactor property. This property has no effect on non-zoomable elements. See {{CSSXRef("-ms-content-zooming")}} for details on zoomable elements.

Starting with Windows 8.1, this property is also supported for touchpad interaction.

This property requires Windows 8 or later.
