The -ms-scroll-limit-x-min CSS property is a Microsoft extension that specifies the minimum value for the {{domxref("Element.scrollLeft")}} property.

{{cssinfo}}

Syntax

Values

{{cssxref("length")}}

The minimum value for the scrollLeft property. If the value is negative, then 0 is used.

Formal syntax

  {{csssyntax}}

Specifications

Not part of any specification.

Remarks

This property has no effect on non-scrollable elements.

The specified behavior of the -ms-scroll-limit-x-min property is only applicable at a {{cssxref("msContentZoomFactor")}} of 1; the behavior of this property is undefined at other zoom factors.

This property requires Windows 8 or later.
