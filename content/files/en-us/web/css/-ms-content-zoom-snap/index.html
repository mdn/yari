The -ms-content-zoom-snap CSS shorthand property is a Microsoft extension that specifies values for the {{CSSXref("-ms-content-zoom-snap-type")}} and {{CSSXref("-ms-content-zoom-snap-points")}} properties.

{{cssinfo}}

Syntax

The -ms-content-zoom-snap shorthand property is specified as one or both of the following content zoom snap values, in order, separated by spaces.

Values

-ms-content-zoom-snap-type
Value of the {{CSSXref("-ms-content-zoom-snap-type")}} property.
-ms-content-zoom-snap-points
Value of the {{CSSXref("-ms-content-zoom-snap-points")}} property.

Formal syntax

  {{csssyntax}}

Specifications

Not part of any specification.

Remarks

This property has no effect on non-zoomable elements. See {{CSSXref("-ms-content-zooming")}} for details on zoomable elements.

Starting with Windows 8.1, this property is also supported for touchpad interaction.

This property requires Windows 8 or later.
