The ::-ms-reveal CSS pseudo-element is a Microsoft extension that is used to display a password reveal button for use with a password field created by {{HTMLElement("input/password", '<input type="password">')}}. The user presses the button to reveal the actual field value rather than asterisks.

The password reveal button is usually displayed at the edge of the password field.

The ::-ms-reveal pseudo-element is specific to Internet Explorer 10 and later.
