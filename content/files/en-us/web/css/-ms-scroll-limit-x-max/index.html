The -ms-scroll-limit-x-max CSS property is a Microsoft extension that specifies the maximum value for the {{domxref("Element.scrollLeft")}} property.

{{cssinfo}}

Syntax

Values

auto

The maximum value for the scrollLeft property is equal to {{domxref("Element.scrollWidth")}}.

{{cssxref("length")}}

The maximum value for the scrollLeft property.

Formal syntax

  {{csssyntax}}

Specifications

Not part of any specification.

Remarks

This property has no effect on non-scrollable elements.

The specified behavior of the -ms-scroll-limit-x-max property is only applicable at a {{cssxref("msContentZoomFactor")}} of 1; the behavior of this property is undefined at other zoom factors.

This property requires Windows 8 or later.
