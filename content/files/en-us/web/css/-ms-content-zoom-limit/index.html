The -ms-content-zoom-limit CSS shorthand property is a Microsoft extension that specifies values for the {{CSSXref("-ms-content-zoom-limit-min")}} and {{CSSXref("-ms-content-zoom-limit-max")}} properties.

{{cssinfo}}

Syntax

The -ms-content-zoom-limit shorthand property is specified as one or both of the following content zoom limit values, in order, separated by spaces.

Values

-ms-content-zoom-limit-min
Value of the {{CSSXref("-ms-content-zoom-limit-min")}} property.
-ms-content-zoom-limit-max
Value of the {{CSSXref("-ms-content-zoom-limit-max")}} property.

Formal syntax

  {{csssyntax}}

Specifications

Not part of any specification.

Remarks

This property has no effect on non-zoomable elements. See {{CSSXref("-ms-content-zooming")}} for details on zoomable elements.

Starting with Windows 8.1, this property is also supported for touchpad interaction.

This property requires Windows 8 or later.
