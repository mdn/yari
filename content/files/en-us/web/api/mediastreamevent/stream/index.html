The read-only property MediaStreamEvent.stream returns the {{domxref("MediaStream")}} associated with the event.

Syntax

 var stream = event.stream;

Example

pc.onaddstream = function( ev ) {
                      alert("A stream (id: '" + 
                            ev.stream.id + 
                            "') has been added to this connection.");
                   };
